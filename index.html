# 介绍

  1. 基于Element UI封装多个组件，并最终组合为一个终极组件CRUD（点击查看效果），达到配置化开发。
  2. 提供了大量的工具函数，常量，提高开发效率。
  3. 可扩展性强，大量属性，事件和插槽透传，不用担心定制化工作。
